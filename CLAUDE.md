## Project

AniMemoria — platform where you can watch anime.
Monorepo: pnpm workspaces + Turborepo.
Language: TypeScript throughout (Node.js runtime).

## Documentation Navigation

This is a NestJS + Turborepo monorepo. Documentation is distributed across apps and packages.

**BEFORE starting any task**, consult the table of contents to find relevant docs:

```
@_docs/TABLE_OF_CONTENTS.md
```

Then read **only** the README files listed there that are relevant to the current task.
README locations: `apps/*/README.md`, `packages/*/README.md`

Do NOT read all READMEs — only those whose description in the table of contents matches the task.
